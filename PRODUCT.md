# Stacks Proof Pad

Register: brand

Stacks Proof Pad is a tiny public project for the Stacks Builder Rewards campaign. It helps a builder publish a short proof-of-build memo and pair it with a minimal Stacks smart contract.

Users: hackathon builders, reviewers, and anyone checking whether the project is real.

Tone: fast, concrete, builder-first.

Principles:
- The first screen must show a usable artifact, not a marketing page.
- The project should be understandable in under 30 seconds.
- The deployment path should stay free: GitHub plus Vercel, Cloudflare Pages, or Netlify.
